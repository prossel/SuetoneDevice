# SuetoneDevice

Code de l'objet Suetone, pour DFRobot Firebeetle 2 ESP32-S3.

Un projet 2024 de [Basile Brun](https://github.com/chap0ng/2023-head-md-future-of-drawing/tree/main/expo) / HEAD-Genève / master Media Design

## Installation

* Arduino IDE
* Board esp32
* Librairie  [DFRobot_AXP313A](https://github.com/cdjq/DFRobot_AXP313A)

Plus d'infos sur https://wiki.dfrobot.com/SKU_DFR0975_FireBeetle_2_Board_ESP32_S3_Advanced_Tutorial#target_9

## Configuration

Tools > Board manager: DFRobot Firebeetle 2 ESP32-S3

Tools > Port: your port

Pour avoir les messages sur le port série:

Tools > USB CDC on boot: Enabled

Tools > Flash size: 16 MB
